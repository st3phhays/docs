---

Chocolatey GUI provides the ability to toggle between light mode (the default selection) and a dark mode within the application.

The following screenshot illustrates where the toggle button is located:

![Show where the light/dark toggle is located within the User Interface](/images/chocolatey-gui/user_interface_main-window_action_toggle_mode_1.png "Show where the light/dark toggle is located within the User Interface")

This is what the User Interface looks like when light mode is selected:

![Show what the User Interface looks like when light mode is selected](/images/chocolatey-gui/user_interface_main-window_action_toggle_mode_2.png "Show what the User Interface looks like when light mode view is selected")

This is what the User Interface looks like when dark mode is selected:

![Show what the User Interface looks like when dark mode is selected](/images/chocolatey-gui/user_interface_main-window_action_toggle_mode_3.png "Show what the User Interface looks like when dark mode is selected")

If required, the <Xref title="default selection for light/dark mode" value="default-to-dark-mode" /> can be modified to your personal liking.

> :choco-info: **NOTE**
>
> This button exists when using the entire application
